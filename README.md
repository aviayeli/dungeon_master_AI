# Dungeon Master AI — הרפתקת פנטזיה עברית

משחק הרפתקאות טקסט מבוסס בינה מלאכותית בעברית, המונע על ידי Claude של Anthropic.
הדאנג'ן מאסטר מספר סיפור, מנהל קרבות ומגיב לפעולות השחקן — הכל בעברית, בזמן אמת.

---

## תיאור המשחק

השחקן בוחר שם וגיבור ומתחיל הרפתקה בעולם פנטזיה. בכל תור, השחקן מקליד פקודה או תיאור פעולה,
והדאנג'ן מאסטר (מבוסס Claude) מגיב עם תיאור עשיר, מרתק ועקבי. המשחק כולל מצב שחקן מלא
(חיים, מיקום, מלאי), שמירה וטעינה אוטומטית, ובקרת עלויות API בזמן אמת.

## ארכיטקטורה

```
main.py              <- נקודת כניסה, דיאלוג התחלה
|
+-- src/
|   +-- config.py       <- משתני סביבה וקבועי תצורה
|   +-- state.py        <- Player, GameState — שמירה/טעינה ל-JSON
|   +-- gatekeeper.py   <- בקרת תקציב API (FinOps)
|   +-- tools.py        <- WebSearchTool לאנתרופיק
|   +-- agent.py        <- DungeonMasterAgent — לולאת inference + קיצוץ היסטוריה
|   +-- sdk.py          <- DungeonMasterSDK — ממשק non-blocking עם threading
|   +-- gui_panels.py   <- LeftPanel (שחקן) + RightPanel (סטטיסטיקה)
|   +-- gui_center.py   <- CenterPanel — בועות צ'אט + אנימציית טעינה
|   +-- gui_main.py     <- App(tk.Tk) — חלון ראשי + שורת קלט
|
+-- tests/
    +-- test_state.py      <- 9 בדיקות ל-State Manager
    +-- test_gatekeeper.py <- 9 בדיקות ל-Gatekeeper
    +-- test_sdk.py        <- 11 בדיקות ל-SDK
```

### עקרונות עיצוב

- **Separation of Concerns**: הממשק גרפי מדבר רק עם ה-SDK. ה-SDK מדבר עם ה-Agent. הגרפיקה לא נוגעת ב-API.
- **Non-Blocking UI**: כל קריאה ל-LLM רצה ב-daemon thread. הממשק לעולם לא קופא.
- **FinOps Truncation**: רק 5 האינטראקציות האחרונות נשלחות ל-LLM — מניעת פיצוץ טוקנים.
- **Budget Guard**: ה-Gatekeeper מפסיק את המשחק אם עלות ה-API חורגת מהמגבלה.

## התקנה והפעלה

### 1. שכפל את המאגר

```bash
git clone https://github.com/aviayeli/dungeon_master_AI.git
cd dungeon_master_AI
```

### 2. הגדר משתני סביבה

```bash
cp .env-example .env
# ערוך את .env והכנס את מפתח ה-API שלך:
# ANTHROPIC_API_KEY=sk-ant-...
```

### 3. התקן תלויות

```bash
uv sync
```

### 4. הפעל את המשחק

```bash
uv run main.py
```

> **הרצה ב-WSL / Linux ללא תצוגה גרפית?**
> יש להגדיר X11 forwarding או להשתמש ב-VcXsrv / WSLg.

## פקודות עבריות נתמכות

| כפתור / פקודה | תיאור |
|---|---|
| `עזרה` | שאל את הדאנג'ן מאסטר מה אפשר לעשות |
| `שמור` | שמור את מצב המשחק הנוכחי לקובץ |
| `טען` | טען את המשחק השמור האחרון |
| `מלאי` | הצג את המלאי הנוכחי של הדמות |
| `יציאה` | צא מהמשחק (עם אפשרות שמירה) |

ניתן גם להקליד כל טקסט חופשי בעברית בשדה הקלט ולשלוח עם **Enter** או כפתור **שלח**.
